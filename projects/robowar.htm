Antweight Wedge-Flipper Combat Robot - Design and Project Report

1. Introduction

This project report details the design and development of an antweight (200g) combat robot equipped with a wedge-flipper mechanism. The robot is engineered for robowar competitions, featuring a low-profile triangular wedge for stability and an integrated flipper to topple opponents.

2. Objectives

  • Design a compact, lightweight, and durable combat robot.
  • Integrate a flipping mechanism within the wedge for offensive capability.
  • Ensure compliance with robowar regulations, including size and weight limits.
  • Optimize robot control and maneuverability for effective combat performance.

3. Design Specifications

3.1 Dimensional Constraints

  • Maximum weight: 200g
  • Maximum dimensions: 15cm x 15cm x 15cm
  • Triangular wedge side profile for minimal height

3.2 Materials Used

Chassis and Frame:

  • 3mm polycarbonate sheet (for wedge)
  • 3D-printed or laser-cut acrylic frame
  • Lightweight aluminum strips (for reinforcement)
  • Screws, nuts, and zip ties for assembly

Electronics:

  • Micro gear motors (2x) with wheels
  • 2.4 GHz RC receiver and transmitter
  • LiPo battery (2S, 7.4V, 300-500mAh)
  • Electronic Speed Controllers (ESCs) (2x)
  • Servo motor (high torque, metal gear) for flipper

Tools Required:

  • Soldering iron
  • Screwdriver set
  • Hot glue gun
  • Dremel or laser cutter (for precision cutting)
  • Multimeter (for testing connections)

4. Robot Construction

4.1 Designing the Robot

  • The side profile is a triangular wedge for minimal height.
  • The wedge is reinforced with polycarbonate for durability.
  • The flipper mechanism is powered by a servo motor and integrated into the wedge.

4.2 Assembling the Chassis

  1. Cut the 3D-printed or acrylic chassis parts based on the design.
  2. Attach the motors to the base using screws and zip ties.
  3. Secure the wheels to the motors.
  4. Reinforce the wedge with aluminum strips for strength.
  5. Attach the servo motor and hinge mechanism for the flipper.

4.3 Wiring and Electronics

  1. Connect the ESCs to the motors and battery.
  2. Connect the servo motor to the RC receiver.
  3. Route the wiring properly and secure it using zip ties.
  4. Test motor control and flipper actuation using the RC transmitter.

5. Testing and Optimization

  1. Secure all components within the chassis.
  2. Perform weight checks to ensure compliance with the 200g limit.
  3. Test drive the robot in a practice arena.
  4. Fine-tune motor speeds, flipper response, and wedge angles.
  5. Conduct durability tests to ensure the robot withstands impacts.

6. Competition Strategies

  • Aggressive Driving: Utilize speed to push opponents out of the arena.
  • Flipping Maneuver: Position the wedge correctly to maximize flipping efficiency.
  • Defensive Tactics: Use the wedge to absorb hits and deflect attacks.
  • Battery Management: Ensure full charge before matches.
  • Quick Repairs: Keep spare parts like extra wheels, servo motors, and batteries ready.
